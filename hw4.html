Table

Lists Usage
Unordered List An unordered list is used to show a list of items that does not need to be in any particular order and is denoted with bullets, using the tag <ul>
Ordered List An ordered list is used to show a list of items that need to be in a specific order and is denoted with numbers, using the tag <ol>
Definition List A definition list is used when items come in pairs and is depicted with indentation in relation to the term that is being defined, using the tag <dl>
